## CONEXIÓN Y NAVEGACIÓN  

- **Conectarte al servidor remoto**
    ```
    ssh -i clave.pem ubuntu@TU_IP
    ```
  Te conectas a tu servidor remoto<br><br>

- **Ver en qué carpeta estás**
    ```
    pwd
    ```
  "print working directory"<br><br>

- **Listar archivos**
    ```
    ls
    ls -l
    ```
  Ver contenido de carpetas<br><br>

- **Entrar a carpeta**
    ```
    cd nombre_carpeta
    ```

- **Volver atrás**
    ```
    cd ..
    ```
